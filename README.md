# Sakshi Sonwane - UI/UX Designer Portfolio

## Overview

This is the personal portfolio website for **Sakshi Sonwane**, a talented UI/UX Designer and Frontend Developer. The site showcases her professional journey, design philosophy, skills, notable projects, education, and work experience.

Key features:

- **Responsive design** optimized for desktop and mobile devices
- **Modern UI** with smooth scrolling, backdrop blur effects, and hover animations
- **Tailwind CSS v4** powered styling for rapid development and customization
- **Portfolio showcase** featuring high-fidelity mockups of real projects including real estate websites, mobile apps, healthcare landing pages, and e-commerce platforms
- **Clean navigation** with anchor links for seamless single-page navigation

## How to Run

### Option 1: Live Server (Recommended for Development)

1. Open terminal in the project root (`c:/Users/Asus1/Desktop/PROJECTS`)
2. Install Live Server globally if not installed:
   ```
   npm install -g live-server
   ```
3. Start the development server:
   ```
   live-server src
   ```
4. Open `http://localhost:8080` in your browser

### Option 2: Direct File Opening

1. Double-click `src/index.html` to open in your default browser
2. Or drag `src/index.html` into any modern web browser

### Option 3: VS Code Live Server Extension

1. Install the \"Live Server\" extension in VS Code
2. Right-click `src/index.html` → \"Open with Live Server\"

## Development Workflow

To customize or extend the portfolio:

1. **Edit content**: Modify `src/index.html` directly
2. **Style changes**: Edit your Tailwind config/source files, then rebuild:
   ```
   npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
   ```
3. **Add images**: Place in `src/` folder and update `index.html` references
